# Harness Router Examples

## SIMPLE

- "现在几点" -> run time/date command, no harness.
- "解释这个函数做什么" with one pasted snippet -> answer directly.
- "把这个 README 里的错别字改掉" -> simple if one typo and no rule/doc structure impact.

## HARNESS

- "修一下登录 bug" -> read harness, then backend/frontend/debugging route.
- "优化前端" -> read harness, then `frontend-task-router`.
- "新增一个 skill" -> read harness, then `skill-creator`.
- "把这些历史文档整理一下" -> read harness, then migrate to practice/feedback/references/works.
- "这次 review 后记住别再犯" -> read harness, update feedback/prompts/scripts as appropriate.
- "改 API" -> read harness, development-pipeline/backend, contracts and validation.
